
# Hilfe

RockHopper-Exkursionen sollten (hoffentlich!) recht intuitiv zu bedienen sein, aber diese Seite hilft dabei, die Funktionsweise im Detail zu erklären.

## 3D-Steuerung

Verwenden Sie die folgenden Steuerungen, um den 3D-Viewer zu navigieren.

| Control |     Action     |
|----------|----------|
| `Left-mouse` | Ansicht drehen   |
| `Right-mouse` | Ansicht verschieben   |
| `Scroll` | Zoom   |
| `Double-click` | Rotationszentrum festlegen oder angeklickte Annotation löschen  |
| `Shift+Left-click` | Ansicht verschieben |
| `Ctrl+Left-click` | Punkt zur aktuellen Annotation hinzufügen |


## Visualisierungssteuerung

**Verwenden Sie die Schaltflächen oben links in der 3D-Ansicht, um den Darstellungsstil zu ändern**.
Die obere Reihe ändert die Farbabbildung, mit der die Punktfarben definiert werden.
Die zweite Reihe kann verwendet werden, um bestimmte Untergruppen der Punkte hervorzuheben oder auszublenden (basierend auf einer zugrunde liegenden Klassifikation).

Dies kann nützlich sein, um verschiedene Zeitpunkte in einem Zeitreihendatensatz zu visualisieren oder bestimmte Bereiche einer Punktwolke hervorzuheben.
